Разработай архитектуру высокопроизводительного игрового сервера на Go для онлайн-MOBA (клиент на Phaser 3).

   1. Network: Используй библиотеку pion/webrtc (или geckosio/go-geckos) для реализации UDP-подобной связи через WebRTC Data Channels.

   4. Features:
   * Reconciliation: Сервер должен возвращать lastProcessedSeq в каждом ответе.
      * Visibility (Fog of War): Напиши функцию, которая фильтрует список игроков в WorldSnapshot для конкретного клиента на основе дистанции обзора.
      * Projectiles: Набросай структуру для летящих снарядов (Position, Velocity) и их обновления в цикле.
